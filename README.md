# Cargame

[**Live Demo**](https://kyle.graehl.org/cargame)

A kid-friendly top-down racecar game.

## Development

### Setup

```bash
npm install
```

### Dev Server

Start the development server:

```bash
npm run dev
```

### Build

Build for production:

```bash
npm run build
```

The output will be in the `dist` directory.
